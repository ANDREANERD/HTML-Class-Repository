Sarfdas

Application
“Science brings us the knowledge to find real solutions.”

View, validate and analyze deforestation for Palawan

SERVIR-Mekong developed a system to detect forest disturbances for the Prey Lang Wildlife sanctuary in collaboration with Greening Prey Lang. The system uses Synthetic Aperture Radar (SAR) technology. A main advantage of the system is that it is not impeded by persistent cloud cover. The system generates alerts for forest disturbances using artificial intelligence. The forest disturbance alert is validated using optical data in Collect Earth Online (CEO; collect.earth). Collect Earth Online is a custom built, open-source, satellite image viewing and interpretation system developed by SERVIR and the FAO as a tool for use in projects that require land cover and/or land use reference data. However the alerts are generated in Google Earth Engine (GEE) and the data transfer between the systems needs to be done manually. Conservation International has expressed interest to replicate the SAR alert system for the southern part of Palawan, the Philippines combined with an automated project creation in CEO. .
